# AI Higress 前端UI设计示例

## 设计原则

基于已有的ink风格UI组件库，设计简洁、现代、具有东方美学特色的管理界面。

## 页面设计说明

### 1. 登录页面 (/login)

#### 设计要点
- 居中布局，使用BrushBorder组件作为容器
- 简洁的表单设计，包含用户名/邮箱和密码输入
- 登录按钮使用InkButton组件
- 底部提供注册链接
- 响应式设计，适配移动设备

#### 使用组件
- BrushBorder（容器）
- InkButton（按钮）
- 标准HTML表单元素

#### API集成
- POST `/auth/login` - 提交登录表单
- 成功后存储token并重定向到仪表盘
- 错误时显示Toast提示

### 2. 仪表盘概览页面 (/dashboard/overview)

#### 设计要点
- 顶部导航栏显示用户信息和主要操作
- 状态卡片区域展示系统核心组件状态
- 提供商网格展示所有活跃提供商
- 指标概览区域展示关键性能数据
- 快速操作区域提供常用功能入口

#### 使用组件
- BrushBorder（卡片容器）
- InkButton（各种操作按钮）
- Lucide React图标
- 状态指示器

#### API集成
- GET `/providers` - 获取提供商列表
- GET `/system/status` - 获取系统状态（管理员）
- GET `/providers/{id}/metrics` - 获取提供商指标

### 3. API密钥管理页面 (/dashboard/api-keys)

#### 设计要点
- 顶部标题和创建按钮
- 密钥列表使用卡片或表格布局
- 每个密钥显示名称、前缀、创建时间、有效期
- 提供复制、编辑、删除操作按钮
- 创建/编辑表单在模态框中展示

#### 使用组件
- BrushBorder（密钥卡片）
- InkButton（操作按钮）
- 模态框组件（创建/编辑表单）
- 密钥强度指示器

#### API集成
- GET `/api-keys` - 获取API密钥列表
- POST `/api-keys` - 创建新API密钥
- PUT `/api-keys/{id}` - 更新API密钥
- DELETE `/api-keys/{id}` - 删除API密钥

## 交互设计规范

### 状态反馈
- 加载状态：按钮禁用并显示加载文本
- 成功操作：显示绿色Toast提示
- 错误操作：显示红色Toast提示
- 警告信息：显示黄色Toast提示

### 确认操作
- 删除操作：二次确认对话框
- 重要配置变更：确认对话框
- 批量操作：确认并显示影响范围

### 表单验证
- 实时验证：输入时即时反馈
- 提交验证：提交时检查所有字段
- 错误提示：具体字段错误高亮显示

## 响应式设计

### 断点定义
- 移动设备：<768px
- 平板设备：768px-1023px
- 桌面设备：≥1024px

### 适配策略
- 移动端：单列布局，折叠菜单
- 平板端：两列布局，简化导航
- 桌面端：完整布局，丰富交互

### 组件适配
- 表格：移动端转为卡片列表
- 表单：移动端垂直排列
- 图表：移动端简化显示
- 导航：移动端折叠为汉堡菜单

## 可访问性设计

### 键盘导航
- 所有交互元素支持Tab键导航
- 明确的焦点指示器
- 合理的Tab顺序

### 屏幕阅读器
- 语义化HTML结构
- 图片alt文本
- 表单标签关联
- ARIA属性补充

### 颜色对比
- 文本与背景对比度≥4.5:1
- 重要信息不仅依赖颜色区分
- 考虑色盲用户需求

## 性能优化

### 图片优化
- 使用适当格式（WebP、AVIF）
- 响应式图片
- 懒加载非关键图片

### 代码分割
- 路由级别代码分割
- 组件懒加载
- 第三方库按需引入

### 缓存策略
- 静态资源长期缓存
- API数据合理缓存
- 离线功能支持

## 设计系统

### 色彩规范
- 主色调：深蓝色（用于主要元素）
- 辅助色：灰色（用于次要元素）
- 强调色：红色/紫色（用于重要操作）
- 中性色：黑/白/灰（用于文本和背景）

### 字体规范
- 标题：serif字体，具有东方美学特色
- 正文：无衬线字体，保证可读性
- 代码：等宽字体，便于阅读

### 间距规范
- 基础单位：4px
- 组件间距：8px、16px、24px、32px
- 页面边距：24px、32px
- 内容区域最大宽度：1200px

### 动效规范
- 过渡时间：200ms、300ms
- 缓动函数：ease-out
- 动画类型：透明度、位移、缩放
- 微交互：悬停、点击、焦点状态
